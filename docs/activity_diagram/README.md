# Activity Diagram - Use Case trọng tâm

Thư mục này chứa PlantUML activity diagram cho các use case chính trong [docs/USE_CASE_DETAILS.md](../USE_CASE_DETAILS.md).

## Danh sách file

- `uc01-login-logout.puml`: Đăng nhập/Đăng xuất
- `uc02-master-data-management.puml`: Cấu hình dữ liệu nền (Master Data)
- `uc03-client-management-ocr.puml`: Quản lý khách hàng (kèm OCR AI)
- `uc04-create-reservation.puml`: Tạo đặt phòng
- `uc05-update-extend-cancel-reservation.puml`: Điều chỉnh/Gia hạn/Hủy đặt phòng
- `uc06-whiteboard-dispatch.puml`: Whiteboard điều phối phòng/lịch ở
- `uc07-checkin-directcheckin-smartlock.puml`: Check-in + Direct check-in + Smart Lock
- `uc08-checkout-cleaning-automation.puml`: Check-out + tự động hóa tạo task dọn dẹp
- `uc09-invoice-payment.puml`: Lập hóa đơn & ghi nhận thanh toán
- `uc10-cleaning-shift.puml`: Lập ca/Phân công/Cập nhật trạng thái dọn dẹp
- `uc11-dashboard-reporting.puml`: Dashboard & Báo cáo (kèm gửi định kỳ qua n8n)
- `uc12-ai-assistant.puml`: Trợ lý AI tra cứu và tóm tắt

## Gợi ý render

Bạn có thể dùng PlantUML extension trong VS Code để preview từng file `.puml`.
